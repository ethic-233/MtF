跨性别女性医疗和顺性别女性是有差异的。顺性别女性医疗一般无需特别考虑激素所致血栓和凝血问题，跨性别女性是要每年去复查凝血功能（以及其他一大堆检查）的。一般的医学综述不会特意提跨性别医疗。
